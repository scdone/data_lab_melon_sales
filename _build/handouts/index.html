Replacing the Accounting Team

Introduction

Imagine that at the company you work for, the accounting team needs to be automated. To quickly replace their function within the company, the CEO has written the python script accounting.py.

Can you help rewrite it so that it’s more understandable? You’ll need to read the code and break out each part into it’s own function with a well-chosen name.

Here’s what the code does:

  • Counts the amounts of each type of melon that were sold

  • Calculates the revenue from those melon tallies

  • Separates sales into online sales and phone sales

  • Produces fancy report to summarize the info for our CEO

To Do

  • Read through the code and understand what it’s doing.

  • Refactor code to make it more readable and reduce repetition. Functions FTW!

  • Add comments explaining what each line of code is doing.

Hint: Which sections can be broken out into functions? Can you think of better names for any of the variables? How would you reduce the use of global variables? Make use of white space to increase readability.
